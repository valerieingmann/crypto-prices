# Running Locally

```
npm install
touch .env
cd client && npm install
```

Add `API_KEY = "your api key"` to `.env`

```
npm run server

// in another terminal
npm run client
```

App runs on Port 3000 and API runs on Port 8080.

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).
